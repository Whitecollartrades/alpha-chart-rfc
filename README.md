# AlphaChart-RFC: Real-Time Trading Analysis with React Financial Charts & Alpha Vantage

Redesigned TradingView-like app using React Financial Charts (RFC) for candlesticks + RSI.

## Features
- Candlestick charts with RSI pane
- User-inputted Alpha Vantage API key (saved locally)
- Real-time updates via polling (15s)
- Dark mode toggle
- Responsive design

## Setup
1. Clone: `git clone https://github.com/yourname/alpha-chart-rfc.git`
2. Install: `npm install`
3. Run: `npm run dev`
